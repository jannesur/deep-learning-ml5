# deep-learning-ml5
Bilderkennung (Klassifikation) mit dem ml5 Framework

# Quelle der Bilder:
Alle verwendeten Bilder wurden von Pixabay bezogen und sind gemäß der Pixabay-Lizenz frei zugänglich und ohne Urheberangabe nutzbar.

# Projekt lokal starten:
- Terminal: python3 -m http.server 8000
- Web: http://localhost:8000/

# Projekt abrufbar über GitHub Pages:
https://jannesur.github.io/deep-learning-ml5/

# Technologien
- ml5.js
- MobileNet
- Chart.js
- HTML, CSS, JavaScript
